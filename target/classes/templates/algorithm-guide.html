🔄 Insertion Sort

How it works:

Think of sorting playing cards in your hand:

  1. Start with the 2nd card (position 1)
  2. Compare it with cards to the left
  3. Shift larger cards to the right to make space
  4. Insert the card in the correct position
  5. Repeat for all remaining cards

Example: [5, 2, 4, 6, 1, 3]

Take 2, shift 5 right → [2, 5, 4, 6, 1, 3]

Take 4, shift 5 right, insert 4 → [2, 4, 5, 6, 1, 3]

⚡ Quick Sort

How it works:

Divide and Conquer approach:

  1. Pick a "pivot" element (usually the last one)
  2. Move all smaller elements to the left of pivot
  3. Move all larger elements to the right of pivot
  4. Now pivot is in its final correct position
  5. Repeat the process for left and right parts

Example: [64, 34, 25, 12, 22, 11, 90] (pivot = 90)

After partition: [64, 34, 25, 12, 22, 11] + [90] + []

90 is now in correct position, sort left part recursively

🔀 Merge Sort

How it works:

Divide, Sort, and Merge:

  1. Divide the array into two equal halves
  2. Keep dividing until you have single elements
  3. Merge two sorted halves by comparing elements
  4. Always pick the smaller element first
  5. Continue merging until you get the full sorted array

Example: [64, 34, 25, 12]

Divide: [64, 34] and [25, 12]

Divide more: [64], [34] and [25], [12]

Merge: [34, 64] and [12, 25]

Final merge: [12, 25, 34, 64]
